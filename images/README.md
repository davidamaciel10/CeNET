# Banco de Imágenes CeNET

Guardá aquí las imágenes de los cursos para usarlas en las cards de CeNET (Moodle).

## Cómo usarlas

Después de subir a GitHub, la URL de cada imagen es:
`https://raw.githubusercontent.com/davidamaciel10/cenet/main/images/NOMBRE_ARCHIVO.jpg`

## Convención de nombres

- Usar el nombre del curso en minúsculas, sin espacios (reemplazarlos por guiones bajos)
- Ejemplos:
  - `ciudadania_digital.jpg`
  - `intro_python.jpg`
  - `machine_learning.jpeg`

## Formatos aceptados

JPG, JPEG, PNG, WEBP — preferir JPG/JPEG por el peso.
Tamaño recomendado: **800 × 450 px** (16:9), peso máximo ~150 KB.

## Notas

- En la versión para **INET** (inet.edu.ar) las imágenes no se muestran — no hace falta completar el campo.
- En la versión para **CeNET** (cenet.inet.edu.ar/Moodle) la imagen aparece en la parte superior de cada card.
